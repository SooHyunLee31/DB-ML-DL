# 중간 발표 피드백 사항 
1. 주제선정배경 
: 기술이 없냐? 정보 비대칭성 부분에서
: 서비스종료라는 부분에 자세히 적기(방산기업인지?)
: 투자부문이랑 기술격차 부문을 강조 -> 내용 좀 요약해라

2. 국내외 기업 선정 기준
: 매출기준 -> 출처남기기, 몇년 몇월 기준인지
: 사업보고서 분기 확인

3. 주제선정 및 분석 목표
: 이 부분이 앞쪽으로 가는게 더 맞을거 같다
: 이 프로젝트의 배경이라 바꾸기
: 그 다음으로 대상이 3번째로 나오게끔

4. 분석데이터 설명
: 데이터 수집의 설명(어떤데이터 인가)
: 국내외 빼고 방산기업 데이터 라고 바꾸기
: 국내는 어디 해외는 어디인지
: 기준 분류을 표로 설명하기

5. 데이터 수집 과정
: 4번이랑 동일한 부분
: 수집방법을 적기, 출처 남기기
: 이미지의 제목을 위에 적으면 좋겠다

데이터 예시
: 국내에서 뭘 했고 결과 보여주기
: 마찬가지로 해외도

6. 데이터 전처리
: 데이터 정제(널값지우거나 등)->데이터 축소(컬럼)->병합->변화(컬럼변화)
: 컬럼 변환한거 등등 자세히 적기
: 변경된거 사례를 들어서 

7. 데이터 분석

8. 시각화
: 설명 달기

9. 대시보드
: UI 뭔지 설명

10. 향후
: 기대효과, 한계점, 출처 넣기
